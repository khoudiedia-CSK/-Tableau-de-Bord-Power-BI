# Tableau de Bord Power BI - Ventes

## Description
Ce projet contient un tableau de bord Power BI permettant de suivre les **ventes** et le **chiffre d’affaires**.  
Le tableau de bord inclut des KPI, des graphiques d’évolution des ventes par jour et une analyse des ventes par produit.

---

## Contenu du dépôt
- `TableauDeBord.pbix` : fichier Power BI avec toutes les mesures et visualisations.
- `Images/` : capture d’écran du tableau de bord.
- `README.md` : ce fichier.

---

## Visualisations incluses

### 1. KPI
- **Total_CA** : chiffre d’affaires total (18K sur l’exemple).  
- **Total_Ventes** : nombre total de ventes.  

### 2. Graphique en colonne (vertical)
- **Somme du Montant par Jour**  
- Permet de suivre l’évolution des ventes quotidiennement.

### 3. Graphique en barre (horizontal)
- **Somme du Montant par Produit**  
- Permet d’identifier les produits les plus vendus (Produit B, C, D, A).  

### 4. Tableau détaillé
- Affiche les ventes par jour, trimestre, mois et année.  

---

## Instructions
1. Télécharger le fichier `Ventes.pbix`.  
2. Ouvrir le fichier avec **Power BI Desktop**.  
3. Interagir avec les filtres et visualisations pour analyser les ventes.

---

## Technologies
- **Power BI Desktop**
- **DAX** pour les mesures (`Total_CA`, `Total_Ventes`)
